# Hermes Shared Skills

Shared skills are reusable single-file Markdown procedures for profiles.

- `generate-handoff/SKILL.md` - canonical cross-session handoff shape.
- `agency/` - converted Agency catalog workflows. See `agency/CATALOG.md` for the full 184-row classification matrix and profile opt-ins.

Keep shared skills small and role-neutral. Profile-specific behavior belongs in `hermes/profiles/<profile>/skills/`.
