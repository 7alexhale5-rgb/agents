# atlas-ceo skills

Atlas has profile-local skills because CEO operating advice is Atlas's core
job, not a shared utility.

- `source-packet-triage.md` checks whether source signals are fresh, missing,
  or contradictory before Atlas recommends anything.
- `weekly-ceo-brief.md` turns a verified source packet into a one-page CEO
  brief for Alex.
- `business-scorecard-brief.md` turns PFOS business scorecard signals into
  priorities, non-priorities, and risks.
- `decision-memo.md` classifies one-way/two-way door risk and names approval
  gates.
- `weekly-ceo-operating-loop.md` keeps Atlas's weekly cadence narrow and
  evidence-gated.
- `approval-proposal-draft.md` prepares proposed-only PFOS action rows for Alex
  approval.

Keep shared behavior in shared skills; keep these files focused on Atlas's
source-grounded CEO operating advisor gate.
