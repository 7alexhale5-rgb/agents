# Communications Triage

Unified direct-provider mail and calendar triage.

V1 is read + propose:

- Gmail: read-only.
- Google Calendar: events read-only + free/busy.
- Microsoft 365: delegated read scopes.
- HostGator: IMAP SSL read-only.
- All mailbox/calendar mutations become proposals.
- No send, no auto-delete, no auto-calendar edit.
