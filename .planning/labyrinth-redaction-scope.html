Labyrinth Plugin-Scoped Redaction

Markdown source: .planning/labyrinth-redaction-scope.md

Summary

Hermes global redaction stays off. Labyrinth forces redaction inside its dashboard plugin process so journey previews and reports do not expose token-shaped strings.

Runtime Breadcrumb

Runtime code lives at ~/.hermes/plugins/hermes-labyrinth/dashboard/plugin_api.py. The versioned decision lives at _meta/decisions/2026-05-20-labyrinth-redaction-scope.md.

Decision Shape

  • No global security.redact_secrets: true.
  • No Hermes core or profile config changes.
  • Set HERMES_REDACT_SECRETS=true inside Labyrinth before agent.redact can import.
  • Keep the prettyfly-fleet plugin-scoped redaction precedent intact.

Verification

Static parse, clean subprocess dummy-token smoke, Labyrinth dashboard health, and PrettyFly Fleet meta check.
