# Marin skills

Five skills, all source-grounded in the marketing vault. Write-class skills write to `~/Projects/marketing/_inbox/marin-readouts/` or an explicitly requested outreach operator packet with an HTML companion.

| Skill                  | When to invoke                                                             | Output                                                                                                           |
| ---------------------- | -------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------- |
| `weekly-review`        | Weekly cadence (or on demand)                                              | Weekly Readout markdown matching `~/Projects/marketing/metrics/weekly-review-template.md`                        |
| `buyer-signal-router`  | A connection accepts, a reply arrives, or Alex asks what the next allowed move is | Proposed next-action memo: signal class, route status, allowed action, ledger update proposal, stop condition     |
| `supervised-dispatch`  | Alex approves up to five LinkedIn prospects for a live supervised dispatch review | Markdown + HTML dispatch packet with route verification, note length, send/hold decision, and stop conditions     |
| `campaign-brief-draft` | New campaign or revision of an existing brief                              | Campaign brief markdown matching `~/Projects/marketing/_templates/Campaign.md` (or the AI Ops Audit brief shape) |
| `aeo-opportunity-scout` | Alex asks how to use Sentinel/AEO beneath Marin or wants buyer-prompt/content-opportunity research | Inbox-only AEO opportunity memo with the 7-part AEO map, source caveats, analytics plan, and no-publish boundary |
| `kill-list-enforce`    | When a request would touch a killed item or violate tool-adoption triggers | 5-line memo: claim → killed item → kill rationale → reopen criteria → recommendation                             |

All skills MUST cite vault file paths for every factual claim. No invented metrics, buyer language, or dates.
